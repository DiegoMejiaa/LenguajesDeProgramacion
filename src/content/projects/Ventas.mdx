La Aplicación de Ventas en Línea es una plataforma diseñada para simplificar y gestionar el proceso de compra y venta de productos de manera eficiente y segura 
Desarrollada en C# como su lenguaje de backend.

Tecnologias Utilizadas:
- DevExtreme
- css
- Html
- C#
- Entity framework
- SQL Server Magagement Studio

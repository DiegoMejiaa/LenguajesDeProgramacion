---
title: "gRPC Chess"
ShortDescription: ": Utilizando la tecnologia gRPC podemos realizar consultas sobre partidas de ajedrez"
---
Este proyecto permite realizar consultas sobre partidas de ajedrez en línea utilizando la tecnología gRPC. A través de un backend eficiente desarrollado en Go, los usuarios pueden acceder a datos detallados de partidas jugadas en plataformas online. Utilizando Microsoft Azure para la infraestructura, Docker para la
contenedorización y Postman para las pruebas.

Lenguajes Utilizados:
- Go

Tecnologias utilizadas:
- Microsoft Azure
- Docker
- Postman
